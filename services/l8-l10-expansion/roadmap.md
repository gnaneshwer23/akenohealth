# L8-L10 Strategic Expansion Roadmap

## L8 Privacy Vault

- Introduce crypto agility and post-quantum migration plan.
- Add zero-knowledge attestations for consent and policy claims.
- Pilot selective homomorphic workflows for high-sensitivity analytics.
- External security and cryptography audit before production use.

## L9 Autonomous Care Agents

- Start with bounded agent domains:
  - medication adherence outreach
  - appointment and discharge follow-up
  - exception triage support
- Require human escalation for uncertain or high-risk decisions.
- Track safety and override telemetry as first-class release criteria.

## L10 Ecosystem and Marketplace

- Release partner API gateway with scoped access tiers.
- Offer cohort discovery and trial matching APIs with governance controls.
- Add usage metering and licensing workflows for commercialization.

## Exit Conditions

- L8: cryptographic control set passes third-party audit.
- L9: autonomous workflows maintain safety thresholds across pilot AMCs.
- L10: at least three revenue-bearing ecosystem integrations live.
