# Multi-AMC Governance RACI

## Councils

- Steering Committee
- Clinical Advisory Board
- Data Governance Council
- Model Safety Review Board
- Compliance and Security Board

## RACI Matrix

| Decision | Steering | Clinical Board | Data Council | Safety Board | Compliance Board |
|---|---|---|---|---|---|
| New pathway launch | A | R | C | C | C |
| Model promotion to production | C | C | C | A/R | C |
| Cross-AMC data contract change | C | C | A/R | C | C |
| High-severity clinical incident response | A | R | C | R | R |
| Consent policy change | C | C | C | C | A/R |
| Partner ecosystem API release | A/R | C | C | C | C |

Legend: R = Responsible, A = Accountable, C = Consulted.

## Cadence

- Weekly: cross-functional release readiness
- Bi-weekly: model safety board
- Monthly: compliance evidence and audit review
- Quarterly: strategic portfolio and AMC expansion review
