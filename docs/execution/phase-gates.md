# Phase Gate Checklists

## Gate: Phase 0 -> Phase 1

- KPI thresholds approved by steering committee.
- Service boundaries and ownership model approved.
- Compliance controls mapped to delivery backlog.

## Gate: Phase 1 -> Phase 2

- At least two AMC connectors passing schema conformance tests.
- Consent-aware authorization enforced for protected APIs.
- Unified patient context endpoint meets latency SLO.

## Gate: Phase 2 -> Phase 3

- Silent-mode model validation completed.
- XAI coverage above threshold.
- Safety board sign-off for live rollout pathways.

## Gate: Phase 3 -> Phase 4

- Live pathways demonstrate operational improvement.
- Monitoring and risk alerting precision target met.
- Clinical governance incidents within tolerance limits.

## Gate: Phase 4 -> Phase 5

- Federated rounds stable across active AMCs.
- Cross-site uplift and fairness checks passed.
- Population intelligence dashboards adopted by site leads.

## Expansion Gate (L8-L10)

- Privacy-vault controls externally audited.
- Autonomous agent workflows stay within safety boundaries.
- Ecosystem APIs enabled with licensing and audit controls.
