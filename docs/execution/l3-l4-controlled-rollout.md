# L3-L4 Controlled Pathway Rollout

## Initial Pathways

- Deterioration risk (medical wards)
- Medication interaction risk (polypharmacy cohort)
- Capacity-aware triage recommendation

## Rollout Pattern

1. Shadow mode for 2-4 weeks.
2. Clinical review only (no autonomous action).
3. Limited live actions with mandatory clinician sign-off.
4. Expand by service line after safety gate pass.

## Required Evidence

- Model performance report (calibration + discrimination + subgroup fairness).
- Explainability coverage report.
- Override and escalation audit logs.
- Incident simulation and rollback drill completion.
